# Club Info — Formations en développement web

Ce dépôt regroupe des formations sur le développement web organisées en modules indépendants (ex : `cours1/`). L'objectif est de fournir des parcours clairs, exercices et corrigés pour des apprenants débutants à intermédiaires.

Structure
---------

- `cours1/` — Cours d'initiation au développement web (HTML, CSS, JS de base)

Comment naviguer
-----------------

1. Ouvrir le dossier d'un cours (ex : `cours1/`).
2. Lire `README.md` du cours pour l'objectif et l'organisation.
3. Consulter `syllabus.md` pour le plan détaillé.
4. Faire les exercices dans `exercises/` et consulter les solutions.
