# Contribuer au dépôt

Merci de votre intérêt pour améliorer ces formations ! Voici un guide rapide pour contribuer.

Processus recommandé
--------------------

1. Forker le dépôt et créer une branche thématique (`feature/ajout-exo`, `fix/typo`, ...).
2. Respecter la convention de structure des cours :
   - `coursX/README.md`
   - `coursX/syllabus.md`
   - `coursX/exercises/`
3. Ajouter des fichiers explicatifs en français et tests/solutions si possible.
4. Ouvrir une Pull Request décrivant les changements et le contexte pédagogique.

Standards de qualité
--------------------

- Rédiger les README en français clair.
- Préférer des exemples courts et fonctionnels.
- Mettre des liens relatifs (pas d'URL absolues vers du contenu non stable).

Reporting de bugs
-----------------

Créer une issue en décrivant :
- le cours concerné
- la page/fichier incriminé
- une proposition de correction
