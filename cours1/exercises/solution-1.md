# Solution — Exercice 1

Exemple de `ex1.html` minimal :

```html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Exercice 1</title>
</head>
<body>
  <header>
    <h1>Mon premier site</h1>
  </header>
  <main>
    <article>
      <p>Ceci est un paragraphe d'exemple pour l'exercice 1.</p>
    </article>
  </main>
  <footer>
    <p>Contact : prenom.nom@example.com</p>
  </footer>
</body>
</html>
```

Explications : utiliser des balises sémantiques améliore l'accessibilité et le référencement.
