# Exercice 1 — Structure HTML

Objectif : créer une page HTML simple respectant la structure sémantique.

Consignes :

1. Créez un fichier `ex1.html`.
2. Ajoutez une entête (`header`) avec le titre du site.
3. Ajoutez une section principale (`main`) avec un article contenant un paragraphe.
4. Ajoutez un pied de page (`footer`) avec vos coordonnées fictives.

Critères de réussite : la page doit être valide HTML5 et bien structurée.
