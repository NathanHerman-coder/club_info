# Cours 1 — Développement web moderne

Résumé
-------

Ce cours d'initiation couvre les fondamentaux du développement web : HTML, CSS, notions de JavaScript, découverte des frameworks frontend et initiation au backend. Le cours est conçu pour un public débutant à intermédiaire.

Objectifs pédagogiques
---------------------

- Comprendre la structure d'une page web (HTML)
- Maîtriser la mise en forme avec CSS et créer des pages responsives
- Découvrir le flux côté client (DOM, événements) et un aperçu de JavaScript
- Présentation de frameworks modernes frontend et concepts backend
- Réaliser un mini-projet fullstack simple

Public cible
-------------

Personnes souhaitant démarrer en développement web sans ou avec peu d'expérience. Les participants doivent avoir un ordinateur et un éditeur de texte.

Prérequis
----------

- Savoir utiliser un ordinateur et un navigateur web
- (Optionnel) notions de programmation

Organisation du cours
---------------------

Durée estimée : 12 à 16 séances (modules théoriques + pratiques).

- Module 1 : Les bases du web (HTML, CSS)
- Module 2 : Frontend et frameworks
- Module 3 : Backend et APIs
- Module 4 : Projet final

Contenu et fichiers
-------------------

- `index.html` : page d'accueil du cours
- `syllabus.md` : programme détaillé
- `exercises/` : exercices pratiques et corrigés

Comment suivre ce cours
-----------------------

1. Lire le syllabus pour connaître le plan.
2. Suivre les séances et compléter les exercices.
3. Proposer des améliorations via des issues ou des pull requests (voir `../CONTRIBUTING.md`).
