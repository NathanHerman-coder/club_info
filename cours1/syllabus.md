# Syllabus — Cours 1 : Développement web moderne

Module 1 — Les bases du web (4 séances)
- Séance 1 : Introduction au web, architecture client/serveur, HTTP
- Séance 2 : HTML — structure de pages, balises sémantiques
- Séance 3 : CSS — sélecteurs, box model, layout (Flexbox)
- Séance 4 : Responsive design, bonnes pratiques d'accessibilité

Module 2 — Frontend et frameworks (4 séances)
- Séance 5 : Introduction à JavaScript — variables, fonctions, DOM
- Séance 6 : Manipulation du DOM, événements, AJAX/fetch
- Séance 7 : Concepts de frameworks (composants, état)
- Séance 8 : Atelier rapide avec un framework (ex: React/Vue) — mise en place d'un petit composant

Module 3 — Backend et APIs (4 séances)
- Séance 9 : Notions de serveur, Node.js et Express (ou équivalent)
- Séance 10 : Routes, middlewares, gestion des formulaires
- Séance 11 : Bases de données (concepts) et connexion simple
- Séance 12 : Authentification basique et sécurité

Module 4 — Projet final (4 séances)
- Séance 13 : Conception du projet, maquettage
- Séance 14 : Développement frontend
- Séance 15 : Développement backend et intégration
- Séance 16 : Présentation et corrections

Ressources recommandées
- MDN Web Docs (HTML/CSS/JS)
- Documentation officielle du framework choisi

Évaluation
- Exercices pratiques après chaque module
- Projet final évalué sur fonctionnalité, propreté du code et documentation
